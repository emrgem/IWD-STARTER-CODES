CSS Units Practice Lab

Pixels (px)

Pixels (px) are an absolute unit, meaning they represent a fixed physical size on the screen. In this exercise, we are using pixels to set the width and height of a div to 200px each.

200px x 200px

Rem (rem)

Rem (root em) is a relative unit that is based on the font size of the root element (usually the html element). In this exercise, we are using rem to set the width and height of a div to 20rem, which will be relative to the root font size.

20rem x 20rem

Em (em)

Em (em) is a relative unit that is based on the font size of the current element. In this exercise, we are using em to set the width and height of a div to 20em, which will be relative to the font size of the div itself.

20em x 20em

Percent (%)

Percent (%) is a relative unit, meaning the size is calculated based on the size of the parent element. In this exercise, we are using percent to set the width and height of a div to 50% of the parent container.

50% x 50%

Viewport Units (vw, vh)

Viewport units (vw, vh) are relative units that are based on the size of the viewport (the visible area of the web page). In this exercise, we are using vw (viewport width) and vh (viewport height) to set the width and height of a div to 50% of the viewport size.

50vw x 50vh
