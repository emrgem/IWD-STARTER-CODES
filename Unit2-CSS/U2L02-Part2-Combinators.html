CSS Combinators:

1. Descendant Selector (space)

nav a { color: blue; text-decoration: none; } <nav class="ex1-nav"> <a href="#">Home</a> <div class="dropdown"> <a href="#">About</a> </div> </nav>

2. Child Selector (>)

.card > p { font-size: 14px; line-height: 1.5; color: #0066cc; background: #e6f0ff; padding: 10px; } <div class="ex2-card"> <p>This will be styled</p> <div> <p>This won't be styled</p> </div> </div>

3. Adjacent Sibling (+)

h2 + p { font-size: 18px; font-weight: bold; color: #cc0066; } <div class="ex3"> <h2>Section Title</h2> <p>This will be bold</p> <p>This won't be bold</p> </div>
