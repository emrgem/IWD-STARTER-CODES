CSS Combinators Lesson

1. Descendant Selector (space)

.descendant-demo p { color: blue; }

Selects ALL paragraphs inside the element with class "descendant-demo", no matter how deeply nested.

This paragraph is blue (direct child)

This paragraph is also blue (nested descendant)

This deep paragraph is still blue!

2. Child Selector (>)

.child-demo > p { color: green; }

Selects ONLY direct child paragraphs of the element with class "child-demo".

This paragraph is green (direct child)

This paragraph stays black (not a direct child)

3. Adjacent Sibling Selector (+)

.adjacent-demo h3 + p { color: red; font-weight: bold; }

Selects paragraph that comes IMMEDIATELY after h3.

Title

This paragraph is red and bold (right after h3)

This paragraph stays normal (not immediately after h3)

4. General Sibling Selector (~)

.sibling-demo h3 ~ p { color: purple; font-style: italic; }

Selects ALL paragraphs that come after h3.

This paragraph stays normal (before h3)

Title

This paragraph is purple and italic

Some div in between

This paragraph is also purple and italic
